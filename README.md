# 2018 두리톤, 두리 웹페이지


## 구현해야 할 것
- 로그인 페이지
-- DB 정보와 맞으면 main 화면으로 이동

- 메인화면
-- 두리에서 진행하는 최근 행사에 관련된 포스터 이미지 슬라이드, 공지사항, 사물함신청, 신문고, 캘린더

- 공지사항
-- 최근 두리에서 진행한 행사의 포스터를 상단에 첨부, 공지사항 게시판

- 사물함신청
-- 사물함 신청, 사물함 양도

- 신문고
-- 문의 사항, 개선 사항등을 적어서 관리자의 이메일로 전송

- 캘린더
-- 두리에서 진행하는 행사를 캘린더로 깔끔하게 정리해서 보여줌
